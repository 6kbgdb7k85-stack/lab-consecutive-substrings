FUNCTION consecutiveSubstrings(string){
    VAR currentString
    VAR subStrings
    FOREACH char in string{
        currentString=char
        subStrings.push(currentString)
        FOREACH remaining character in the string (char2){
            currentString+=char2
            subStrings.push(currentString)
        }
    }
    return subStrings
}

This solution takes each letter of a word and adds every iteration of completing the rest of the word from that character.
To do this requires a nested loop so it has complexity of O(n^2).
Other methods using array methods like split or slice were considered, but the complexity is either the same or worse with such solutions
